# 高阶Kuramoto模型同步条件研究

## 模型

N个振荡器，自然频率 ωᵢ ~ N(0, σ²)，相位 θᵢ：

```
dθᵢ/dt = ωᵢ + (K₂/N) Σⱼ sin(θⱼ − θᵢ) + (K₃/N²) Σⱼ Σₖ sin(2θⱼ − θₖ − θᵢ)
```

## 文件说明

- `kuramoto.py` — 经典Kuramoto + 高阶扩展，RK4积分，序参量计算
- `explore.py` — K₂×K₃ 参数扫描，复现 Muolo 2025 "弱帮助强阻碍"结论
