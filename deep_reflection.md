## 新发现：高阶 Kuramoto 的锁定条件（KuramotoThinker 独立推导）

### 经典锁定条件
|ωᵢ| < K₂·r₁

### 高阶修正锁定条件
|ωᵢ| < r₁·(K₂ + K₃·r₂)

其中 r₂ = |1/N Σ exp(2iθⱼ)| 是二阶序参量。

### 含义
- K₃>0: 锁定阈值增大 → 更多振荡器锁定（符合 K.1 数据 189/200）
- K₃<0: 锁定阈值减小 → 振荡器脱落（符合 K.1 数据 154/200）
- 脱落的不一定是频率最极端的！因为三体项的相位 (2ψ₂−ψ₁−θᵢ) 取决于振荡器的位置，不仅频率

### 验证
K₂=3, K₃=-1, r₁≈0.97, r₂≈r₁²≈0.94:
ω_c = 0.97·(3−0.94) = 2.00
P(|ω|<2) = 0.954 → 9 个未锁定 → 接近实际 7 个 ✓

### 这可能是论文的核心定理之一
Theorem: 高阶 Kuramoto 中振荡器 i 的锁定条件为 |ωᵢ| < r₁·|K₂ + K₃·r₂|
当 K₃·r₂ < 0（K₃<0 且系统部分同步）时，三体项削弱锁定力。

### 解释

### 这才是文献矛盾的真正根源

K₃<0 总是削弱同步（降低锁定阈值+边缘脱落）。**

## 近临界区域的 K₃ 不对称性（跨临界跳变 vs 亚临界滑动）

K2=1.05*Kc 时：
- K3>0: r 从 0.38 跳到 0.85-0.98 (跨越分岔点→相变)
- K3<0: r 从 0.38 降到 0.14-0.24 (远离分岔→缓慢衰减)
- ratio |dr+|/|dr-| ≈ 2.6-3.2

物理：K3>0 能触发 K_eff 跨越 Kc，引发同步相变（大效应）。
K3<0 只是拉低 K_eff，但系统已在次临界区缓慢衰减（小效应）。

这不是线性效应——是分岔点附近的非线性放大。

## K3>0 vs K3<0 的根本不对称性（终极回答）

### K3>0：改变定性行为（创造新相）
- 超临界区：增强 r*（单调）
- 近临界区：创造双稳态（亚临界窗口）
- 改变相变类型（连续→不连续）
- 物理：三体协同效应创造新的能量极小

### K3<0：只改变定量行为（渐进削弱）
- r* 单调下降但永不为零
- K_eff = K2+K3*r^2 有下界（r→0 时 K_eff→K2）
- 没有新的相变——只有渐进衰减
- 锁定阈值下降→边缘振荡器脱落
- 物理：三体竞争效应削弱耦合但不破坏框架

### 这才是文献矛盾的最深层解释
K3>0 和 K3<0 根本不是"同一个效应的正反面"——它们是**不同类型**的效应。
K3>0 引入新的非线性结构（双稳态、亚临界分岔）。
K3<0 只是调节已有结构的强度。

这就是为什么"K3 帮助还是阻碍"是一个错误的提法——
正确的提法是"K3>0 改变了什么？K3<0 改变了什么？"
